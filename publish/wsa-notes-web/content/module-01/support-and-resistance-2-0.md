# Support and Resistance 2.0 (Phase One)

## Overview
- Plotting support (floors) and resistance (ceilings) on higher timeframes is how Cue defines the market19s decision zones before worrying about entries.
- Everything is done on a **line chart** so only closing prices matter, making indecision levels clean and repeatable.
- The method revolves around spotting three behaviorstouch, gap, overlapthat prove both buyers and sellers have respected the area.
- Once the levels are in place, every later module (market structure, confirmations, trendlines) plugs into the same zones.

## Setup
1. **Workspace / tools**
   - Charting platform set to **line chart**.
   - Two horizontal-line tools: **red** for Daily, **dark green** for H4. (Optional: blue "random line" for ad-hoc H1 notes.)
2. **Pre-trade context**
   - Start on the Daily timeframe, zoom out far enough to capture ~5k6k pips of history.
   - Sweep top-to-bottom, logging any area where price respected both sides.
   - Drop to H4 only after Daily anchors are placed.
3. **Annotation standards**
   - Daily levels = red horizontals spanning the full visible range.
   - H4 levels = dark-green horizontals, limited to 13 current zones.
   - Label each level with the scenario (Touch, Gap, Overlap) if you need quick recall.

## Execution Rules
1. **Daily pass (structure anchors)**
   - Work left-to-right on the line chart; every level must act as both support and resistance at least once.
   - Accept only three patterns:
     - **Touch**: price taps the line on both sides (most accurate).
     - **Gap**: line sits between two nearby closes; wicks will often appear on the candle view.
     - **Overlap**: closes straddle the line evenly.
2. **H4 refinement**
   - Stay focused on the current price range; no need to zoom out excessively.
   - Plot **one to three** zones that explain present swings (same three scenarios apply).
   - Keep the spacing even; overlaps must have similar distance on both sides.
3. **Maintenance / add-ons**
   - H1 is no longer gridded with S/R; instead, drop a **random blue trend line** wherever a fresh intraday shelf appears.
   - Leave higher-timeframe lines on the chart for 23 weeks; they remain valid until price decisively trends through and retests from the other side.
   - Invalidate a level if it only reacts on one side or if price churns within it without clean reactions.

## Checklist
- [ ] Daily line chart cleaned, zoomed out, red horizontals placed only where both sides traded.
- [ ] H4 refinement done with no more than three dark-green zones around current price.
- [ ] Optional random blue line added for any fresh H1 shelf that isnt yet reflected upstairs.
- [ ] Screenshots saved (line + candle view) so you can review reactions later.

## Common Mistakes
- Treating single-sided hits as valid levels; if only buyers or only sellers touched it, delete it.
- Overloading the H4 chart with more than three lines, which kills focus and reaction speed.
- Drawing gaps/overlaps that arent centered, leading to crooked zones and unreliable reactions.
- Reverting to H1 grids instead of using the flexible random line to mark intraday pivots.

## Drills / Reps
1. **Daily Sweep Drill** 20 charts: switch to line view, plot red levels, flip to candles, and jot how many tests you see per line.
2. **H4 Compression Drill** Pick a live pair, limit yourself to two green lines, and track whether price respects or ignores them over 3 sessions.
3. **Random-Line Rehearsal** During intraday replay, pause whenever you spot a new H1 pivot, drop a blue line, and log whether it later aligns with Daily/H4 structure.

## Questions to Revisit
- Do my current levels show clear touches/gaps/overlaps on both sides?
- When price is mid-range, which two levels bracket the next likely reaction?
- Did any intraday random lines earn a promotion to H4 or Daily status after multiple tests?
